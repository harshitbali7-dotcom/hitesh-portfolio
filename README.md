# Hitesh Bali Portfolio with Working Samples

This package contains a publish-ready portfolio page and two public PDF work samples.

## Files
- `index.html` - main portfolio page
- `work-samples/ups-fedex-financial-analysis.pdf` - public PDF sample
- `work-samples/swiftserve-marketing-plan.pdf` - public PDF sample

## Important confidentiality note
Do not publish full confidential/client decks unless you have permission. The AminoSense and Syngenta sections on the public portfolio are intentionally sanitized.

## GitHub Pages upload
Upload `index.html` and the `work-samples/` folder to your GitHub Pages repository, then enable Pages from Settings > Pages > Deploy from branch.
